# Faceless VPS

Простая программа для автоматической настройки своей сетевой инфраструктуры на VPS.

---

## Зачем это нужно

Если вы пытались настраивать VPS вручную — вы знаете, сколько это занимает времени:

- инструкции на 50 шагов  
- постоянные ошибки  
- "почему не работает"  
- часы, потраченные на разбор  

Я прошёл через это сам и просто устал делать всё руками.

Поэтому сделал программу, которая делает это автоматически.

---

## Что делает программа

Faceless VPS берёт два ваших сервера:

- входной сервер (ближе к вам)
- выходной сервер (в другой локации)

Дальше:

👉 вы вводите IP, логин и пароль  
👉 нажимаете кнопку  
👉 программа сама всё настраивает  

---

## Что настраивается

На выходном сервере автоматически поднимаются несколько протоколов:

- AmneziaWG 2.0  
- VLESS + XHTTP + REALITY  
- NaiveProxy  
- Hysteria 2  

---

## Что вы получаете

Удобный клиент, в котором можно:

- переключаться между протоколами  
- включать автопереключение при сбоях  
- использовать авторотацию  
- настраивать раздельную маршрутизацию  

---

## Важно

- программа не предоставляет серверы  
- вы используете свои VPS  
- все настройки происходят автоматически  
- в программе есть встроенный HELP с инструкцией  

---

## Как начать

1. Запустить программу  
2. Ввести данные двух VPS  
3. Нажать кнопку настройки  
4. Дождаться завершения
